# 决策记录 (Architecture Decision Records)

## 什么是 ADR？

ADR (Architecture Decision Records) 是一种记录重要决策的轻量级文档格式，帮助团队理解为什么做某个决定。

## 命名规范

文件命名格式：`NNN-简短描述.md`

例如：
- `001-选择React作为前端框架.md`
- `002-采用微服务架构.md`
- `003-使用PostgreSQL数据库.md`
